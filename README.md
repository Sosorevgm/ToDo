Приложение для создания и планирования задач. Данный проект был написан в процессе обучения в школе мобильной разработки Яндекса.

Стек технологий:
- MVVM
- kotlin
- dagger 2
- coroutines
- state/shared flow
- retrofit
- room
- androidx navigation
- work manager
- unit tests (junit)
- android tests (android junit, mockito)
- view binding
- kotlin dsl
